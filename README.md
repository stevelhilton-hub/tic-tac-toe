# Tic-Tac-Toe

A simple tic-tac-toe game built with HTML, CSS, and JavaScript.

## How to Play

1. Open `tic-tac-toe.html` in your browser
2. Two players take turns clicking squares — Player X goes first
3. First to get three in a row wins

## Running the Game

No setup required. Just open the HTML file directly in any browser.
